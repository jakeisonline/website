```bash
npx shadcn@latest add https://jakeisonline.com/registry/scale.json
```

## Usage

```jsx
import { Scale, ScaleStep } from "@/components/ui/scale"
```

```jsx
<Scale>
  <ScaleStep id="unhappy" name="satisfaction" label="Unhappy" />
  <ScaleStep id="meh" name="satisfaction" label="Meh" defaultChecked />
  <ScaleStep id="happy" name="satisfaction" label="Happy" />
</Scale>
```

## Accessibility

Adheres to the [Radio Group WAI-ARIA design pattern](https://www.w3.org/WAI/ARIA/apg/patterns/radio/). At its core, this component is simply a [native radio input](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input/radio) wrapped in a [fieldset](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/fieldset).

### Keyboard interactions

| Key                                                | Action                                                                                                                                                                                                               |
| -------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <kbd>Tab</kbd> / <kbd>Shift</kbd> + <kbd>Tab</kbd> | Moves focus into and out of the `Scale` group.                                                                                                                                                                       |
| <kbd>Space</kbd>                                   | Checks the focused `ScaleStep`, if not already focused.                                                                                                                                                              |
| <kbd>ArrowRight</kbd> / <kbd>ArrowDown</kbd>       | Move focus to the next `ScaleStep` in the group, uncheck the previously focused `ScaleStep`, and check the newly focused `ScaleStep`. If focus is on the last `ScaleStep`, focus moves to the first `ScaleStep`.     |
| <kbd>ArrowLeft</kbd> / <kbd>ArrowUp</kbd>          | Move focus to the previous `ScaleStep` in the group, uncheck the previously focused `ScaleStep`, and check the newly focused `ScaleStep`. If focus is on the first `ScaleStep`, focus moves to the last `ScaleStep`. |

## Component Reference

### Scale

Used to wrap a group of `ScaleStep` components.

### ScaleStep

An individual step within the `Scale` component. Also accepts all [native radio input attributes](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input/radio).

| Prop    | Type   | Description                             |
| ------- | ------ | --------------------------------------- |
| `id`    | string | The id of the `ScaleStep`. Required.    |
| `name`  | string | The name of the `ScaleStep`. Required.  |
| `label` | string | The label of the `ScaleStep`. Required. |
